## Arithmetic Operators

### Addition `+`

::tabs{.w-full}
  :::tabs-item{icon="i-lucide-code" label="Code"}
    ```js
    let x = 5;
    let y = 2;
    let z = x + y;
    console.log(z);
    ```
  :::

  :::tabs-item{icon="i-lucide-terminal" label="Output"}
    ```js
    7 // 5 + 2
    ```
  :::
::

::tabs{.w-full}

    ```js
    let x = 5;

    console.log(z);
    ```
  :::

  :::tabs-item{icon="i-lucide-terminal" label="Output"}
