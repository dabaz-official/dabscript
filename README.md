# Welcome to the repository of DabScript!

## What is DabScript?

**[DabScript](https://script.dabaz.me)** is a no-nonsense, essential **JavaScript** guide for beginners, focusing on core concepts and clear, actionable syntax.

DabScript is inspired by a course from [Zach Gollwitzer](https://www.youtube.com/c/ZachGollwitzer).

## Key Features

- DabScript is perfect for **beginners**.
- DabScript follows a **logical sequence** from start to end.
- You should be able to **build basic web pages** and understand the **basics of JavaScript** after learning with DabScript.
